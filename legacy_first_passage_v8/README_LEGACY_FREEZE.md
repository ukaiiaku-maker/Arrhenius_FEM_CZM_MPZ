# Frozen legacy first-passage implementation

This folder is the unmodified v8-era baseline assembled from the two source archives supplied for this revision:

- `FEM_fatigue(1).zip`: campaign scripts, analysis tools, tests, and project metadata;
- `arrhenius_fracture(5).zip`: the active `arrhenius_fracture` source package.

The original input archives are preserved byte-for-byte in `original_input_archives/`. The expanded source tree is provided for inspection and regression. No v9 moving-process-zone changes are applied inside this folder.

Use the active package one directory above for v9 runs. Use this folder only for historical regression and audit.
