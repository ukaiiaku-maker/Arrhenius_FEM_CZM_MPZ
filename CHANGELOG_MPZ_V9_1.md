# MPZ v9.1 changelog

Relative to v9.0:

- Replaced the provisional four-class calibration workflow with a three-class workflow: ceramic, weakT, DBTT.
- Added robust full R-curve metrics and a 1000 micrometre repeated-growth target.
- Added staged parameter fitting to reduce compensation between elementary barriers and process-zone kinetics.
- Added pre-renewal MPZ diagnostics to the front engine.
- Added DBTT initiation and propagation constraints requiring evolved shielding/blunting.
- Added numerical convergence auditing for the fitted classes.
- Updated fatigue, dwell, and FEM/CZM validation defaults to the three-class table.
- Increased package version from 0.9.0 to 0.9.1.
